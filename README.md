# Проект "Посмотри в окно"
https://github.com/JollyCupcake/posmotri-v-okno-fd

## Описание
Проект создан, чтобы показать, что видит кто-то из команды Практикума, когда смотрит в окно. 

## Механика
В правой части перечислены карточки с городами, в которых живут члены команды. При клике на какую-то из карточек в главной части экрана в плеере начинает отображаться выбранный вид из окна. 

Под плеером есть инпут для поиска вида из окна по названию города. В текущей реализации можно искать только те города, которые уже возвращает сервер в списке справа. 

Важно! Инпут для ввода города чувствителен к регистру, что значит что город "Адлер" с заглавной буквы найдётся, а "адлер" - нет.

### Рефлексия
В этом проекте, в отличие от прошлого, не нужно было создавать разметку с нуля. Напротив, требовалось работать с уже имеющейся разметкой, что было классным опытом.
